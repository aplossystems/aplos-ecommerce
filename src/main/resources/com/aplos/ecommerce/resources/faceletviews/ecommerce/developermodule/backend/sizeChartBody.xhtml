Note The name of the measurement is a column in addition to this value. Number of columns is the number of values to display in each table for each measurement type.
Tolerance *
About Tolerance The size chart atom creates multiple tables if the table would be too wide for its content area. It will split the values into tables with the number of columns specified, however when it gets to the last table you may find you only have one or two columns. Having only two columns in a table can look out of place so tolerance is used. If you have a tolerance of two, then, if there are less than 3 columns - ie, 1 column or two columns - the atom will place these on the previous table and the sizes will adjust to fit. It may be a good idea to set the number of columns one lower than would actually fit when using tolerance. Maximum tolerance is 4.
